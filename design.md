# Diretrizes de Design e Identidade Visual - Marcelo Luz Transportes (V5)

## 1. Conceito Visual
Interface "Industrial Premium": tecnologia, segurança e robustez; glassmorphism e cores sólidas para identificação rápida.

## 2. Paleta de Cores Definitiva
- **Tema Claro (Céu e Aço):** Fundo `#c5d9ed`, texto `#1a2b4b`, cards branco translúcido com bordas prateadas sutis.
- **Tema Escuro (Noite Industrial):** Fundo `#0f172a`, texto `#f8fafc`, acentos `#38bdf8` (sem verde no tema base).

## 3. Componentes de Interface
- **Menu:** Ícone Lucide ao lado de cada item.
- **Cards WhatsApp:** Grid simétrico; **(82) 99949-0684** primeiro; `rounded-2xl`, ícone verde oficial `#25D366`, hover que eleva o card.
- **FAB:** Canto inferior direito, número principal **99949-0684**, mensagem "Gostaria de fazer uma cotação".

## 4. Formulário
- Envio para WhatsApp **(82) 99949-0684**.
- Mensagem: títulos em negrito (`*texto*`), emojis 🏠 casa / 🏢 apartamento.
- Progressive disclosure: endereço após tipo de imóvel.

## 5. UX
- Sem hífens (`-`) como separadores em soluções/copy onde indicado.
- Ícones de serviços coloridos e sólidos.
- Galeria: desktop marquee; mobile swipe.
